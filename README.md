# React-Task
This is a simple React project which demonstrate responsive Modal Component

## How to run it:
- `npm install`
- `npm update`
- `npm start`

## Dependencies: 
    react: 16.12.0,
    react-dom: 16.12.0,
    react-scripts: 3.2.0,
    react-select: 3.0.8,

## About app:
Modal component for requesting report. Function and design according to `react-task-outlook.jpg` & `react-task-outlook2.jpg`
Generate button and result Modal are not styled.
